

## Requirements

* [NodeJs](http://nodejs.org) >= 6.x 
* Local [mongodb](http://mongodb.org) running 


## Install

```sh
$ git clone https://github.com/vasist/JRCigars.git
$ npm install
$ node server.js
```

**NOTE:** 
Make sure mongodb is running on the local
